상수 포인터.

상수 포인터는 선언하는 방식에 따라서 상수화할 수 있는 대상이 두가지 이다.

    1. 포인터 변수를 상수화 -> 포인터 변수가 다른 변수를 가리킬 수 없다
    2. 포인터 변수가 가리키는 변수 상수화 -> 포인터 변수가 가리키는 변수에 다른 값을 대입할 수 없다

    ex) 
        1) 포인터 변수 상수화
            int * const pInt_value = &int_value;
        2) 상수 포인터 선언
            const int * pInt_value = &int_value;
    
    보통 상수 포인터는 함수의 매개변수로 전달된 포인터가 함수 내부에서 읽기 전용으로 사용되는 것을 명시적으로 알려준다.
